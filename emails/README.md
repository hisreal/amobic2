# Amobic Email Templates

These are Mailchimp-ready confirmation templates:

- `property-owner-confirmation.html`
- `partner-confirmation.html`

Required Mailchimp merge tags:

- `FNAME`
- `PTYPE`
- `AREA`
- `UNITS`
- `CATEGORY`
- `LOCATION`

The templates also use Mailchimp's `*|UNSUB|*` unsubscribe tag for compliance.
